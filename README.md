## cli介绍

- 便于用户可以快速的加入到PlatON网络和Alaya网络中，用户可以使用cli工具部署主网，测试网和私有网络的节点；
- 便于管理本地节点，如启动节点，更新节点，数据备份和回滚等操作；
- 便于用户管理本地钱包；
- 便于节点转账，质押，委托，治理和合约等相关的操作，并同时提供在线和离线模式；
- 便于用户查询相关节点和链信息等。



## 进度

当前cli已经完成功能的开发，版本状态为**初步验证测试**。
